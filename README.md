## 📌 Project Overview

An end-to-end Human Resources Analytics project developed to analyze employee data and generate business insights using SQL, Oracle Database, Python (Pandas), and Power BI.

The objective is to simulate a real-world HR Analytics project by designing a database, performing SQL analysis, creating dashboards, and answering business questions through data.

## 🗄 Database

The project uses an HR dataset containing information about employees, departments, positions, salaries, performance, and other workforce-related data.

---

## 🎯 Business Questions

Some of the business questions that will be answered include:

- Which department has the highest average salary?
- Which employees earn above the company average?
- Which department has the largest workforce?
- Which jobs have the highest salaries?
- Which employees have the best performance?
- Which department has the highest payroll cost?
- What trends can be identified from the employee data?
